# 富贵花开 - Fortune Bloom
🌸 AI驱动的富贵风格照片美化微信小程序

## 项目介绍

「富贵花开」是一款面向中老年用户的AI照片美化小程序。用户上传照片后，AI自动生成牡丹富贵、金玉满堂、水墨丹青等中国传统喜庆风格的美照。

### 目标用户
- 40-65岁中老年用户
- 微信日活最高的用户群体
- 喜欢喜庆、华贵、传统审美风格

### 核心功能
- 📷 拍照或选择照片，一键AI美化
- 🎨 5种风格：牡丹富贵、金玉满堂、水墨丹青、祥云仙气、古典年画
- 💾 保存到相册，分享到家庭群/朋友圈
- 💎 VIP订阅：日卡¥6.9 / 月卡¥19.9 / 年卡¥99
- 🆓 免费用户每日3次

### 技术架构
```
fortune-bloom/
├── 前端: 微信小程序原生开发
├── 后端: Node.js + Express
├── AI: 通义万相图像风格化API
└── 支付: 微信支付
```

## 部署指南

### 前端（微信小程序）
1. 打开微信开发者工具
2. 导入项目，选择此目录
3. AppID: `wxaa2ab5ad15566072`
4. 在「开发管理」中配置服务器域名白名单
5. 上传代码并提交审核

### 后端
```bash
cd backend
npm install
cp .env.example .env
# 编辑 .env 填入你的 API Key
npm start
```

### 环境变量
| 变量 | 说明 |
|------|------|
| DASHSCOPE_API_KEY | 阿里云通义万相API Key |
| WECHAT_APPID | 微信小程序AppID |
| WECHAT_SECRET | 微信小程序Secret |
| WECHAT_PAY_MCHID | 微信支付商户号 |

## 获取 API Key

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 开通「通义万相」服务
3. 创建API Key并填入 `.env`
